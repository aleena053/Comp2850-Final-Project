### Comp2850-Final-Project

# Project Overview
The Fitness Planner App is a web application designed to help users plan, track, and monitor their workouts and fitness progress. The platform allows users to log workouts, organise training routines, and review their performance over time.

The goal of this application is to provide a simple and structured way for users to manage their fitness activities, whether they are casual gym-goers or competitive athletes. The system is built to focus on workout tracking, progress monitoring and personalised training organisation.

# Project Objectives
The main objectives of this project are:
- Allow users to log and track workouts
- Enable users to view workout history and performance
- Provide a structured way to plan training routines
- Store and manage fitness data efficiently
- Deliver a user-friendly web interface for fitness tracking

# Key Features
The application will include the following core features:
- User account creation and authentication
- Workout logging and tracking
- Workout history and progress monitoring
- Training routine organisation
- Data storage for workouts and users
